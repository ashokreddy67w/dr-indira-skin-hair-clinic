# Dr. Indira's Skin & Hair Clinic Website

Modern Next.js website for a dermatology clinic located in Guntur, Andhra Pradesh.

## Tech Stack
- Next.js
- TypeScript
- Responsive Design
- SEO Optimised

## Features
- Dermatology-focused landing page
- Service sections
- Google Maps integration
- Click-to-call & WhatsApp
- Structured layout for local clinic conversion
